# Commerce System

간단한 콘솔 기반 커머스 플랫폼입니다.  
사용자는 상품을 조회하고 장바구니에 담아 주문할 수 있으며,  
관리자는 상품을 추가/수정/삭제할 수 있습니다.

---

## 프로젝트 소개

이 프로젝트는 **Java 콘솔 환경에서 동작하는 쇼핑 시스템**으로,  
다음과 같은 기능을 제공합니다.

- 카테고리 기반 상품 조회
- 장바구니 관리
- 주문 처리 및 재고 감소
- 관리자 기능 (상품 CRUD)

---

## 프로젝트 구조
org.zerock
├── Main.java # 프로그램 실행
├── CommerceSystem.java # 전체 흐름 제어 (핵심 로직)
├── Product.java # 상품 엔티티
├── Category.java # 카테고리 관리
├── AdminService # 관리자 기능
├── Customer.java # 사용자 (확장용)
├── Rating.java # 고객 등급
├── CartItem.java # 장바구니 아이템
└── CartService.java # 장바구니 로직


---

## 주요 기능

### 1.  상품 조회
- 카테고리별 상품 목록 출력
- 상품 정보
    - 이름
    - 가격
    - 설명
    - 재고

---

### 2. 장바구니 기능

- 상품 추가
- 동일 상품 수량 증가
- 재고 초과 방지
- 장바구니 목록 조회

---

### 3. 주문 기능

- 장바구니 기반 주문
- 총 금액 계산
- 주문 완료 시:
    - 재고 감소
    - 장바구니 초기화

---

### 4. 관리자 기능

- 상품 추가
- 상품 수정
- 상품 삭제

---
